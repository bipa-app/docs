---

## Request

Retrieve a paginated list of Pix transactions (both inbound and outbound).

### Query parameters

<ParamField query="limit" type="integer" default="20">
  Number of transactions to return (1-100)
</ParamField>

<ParamField query="starting_after" type="string">
  Cursor for pagination. Use the `id` of the last transaction from the previous page.
</ParamField>

<ParamField query="ending_before" type="string">
  Cursor for reverse pagination.
</ParamField>

<ParamField query="customer_id" type="string">
  Filter by customer
</ParamField>

<ParamField query="direction" type="string">
  Filter by direction: `inbound` or `outbound`
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `pending`, `processing`, `completed`, `failed`, `refunded`
</ParamField>

<ParamField query="created_after" type="string">
  Filter transactions created after this ISO 8601 timestamp
</ParamField>

<ParamField query="created_before" type="string">
  Filter transactions created before this ISO 8601 timestamp
</ParamField>

## Response

Returns a paginated list of Pix transaction objects.

<ResponseField name="object" type="string">
  Always `list`
</ResponseField>

<ResponseField name="data" type="array">
  Array of Pix transaction objects
</ResponseField>

<ResponseField name="has_more" type="boolean">
  Whether more results exist
</ResponseField>

<ResponseField name="total_count" type="integer">
  Total number of transactions matching the filters
</ResponseField>

import requests
from decimal import Decimal

    return Decimal(amount) / Decimal(10 ** decimals)

    "https://api.bipa.tech/v1/pix",

    params={
        "customer_id": "cus_a1b2c3d4e5f6",
        "direction": "inbound",
        "status": "completed",
        "limit": 50
    }
)

result = response.json()
total_received = sum(
    int(t["amount"]) for t in result["data"]
)
# Assuming all BRL with 2 decimals
total_brl = Decimal(total_received) / Decimal(100)
print(f"Total received: R$ {total_brl:.2f}")

## Pagination example

Iterate through all transactions for a customer:

```python
def get_all_customer_transactions(customer_id: str):
    transactions = []
    starting_after = None

    while True:
        params = {
            "customer_id": customer_id,
            "limit": 100
        }
        if starting_after:
            params["starting_after"] = starting_after

        response = requests.get(
            "https://api.bipa.tech/v1/pix",
            headers={"X-API-Key": "bipa_prod_..."},
            params=params
        ).json()

        transactions.extend(response["data"])

        if not response["has_more"]:
            break

        starting_after = response["data"][-1]["id"]

    return transactions
```

## Aggregating transactions

Calculate daily totals:

```python
from collections import defaultdict
from datetime import datetime
from decimal import Decimal

transactions = get_all_customer_transactions("cus_abc123")

daily_totals = defaultdict(lambda: {"inbound": Decimal("0"), "outbound": Decimal("0")})

for tx in transactions:
    if tx["status"] == "completed":
        date = datetime.fromisoformat(tx["created_at"].replace("Z", "")).date()
        amount = Decimal(tx["amount"]) / Decimal(10 ** tx["decimals"])
        daily_totals[date][tx["direction"]] += amount

for date, totals in sorted(daily_totals.items()):
    print(f"{date}: In: R$ {totals['inbound']:.2f}, Out: R$ {totals['outbound']:.2f}")
```
