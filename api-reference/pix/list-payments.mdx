## Request

Retrieve a paginated list of Pix payments.

### Query parameters

<ParamField query="limit" type="integer" default="20">
  Number of payments to return (1-100)
</ParamField>

<ParamField query="starting_after" type="string">
  Cursor for pagination. Use the `id` of the last payment from the previous page.
</ParamField>

<ParamField query="ending_before" type="string">
  Cursor for reverse pagination.
</ParamField>

<ParamField query="customer_id" type="string">
  Filter by customer
</ParamField>

<ParamField query="direction" type="string">
  Filter by direction: `inbound` or `outbound`
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `pending`, `processing`, `completed`, `failed`, `refunded`
</ParamField>

<ParamField query="created_after" type="string">
  Filter payments created after this ISO 8601 timestamp
</ParamField>

<ParamField query="created_before" type="string">
  Filter payments created before this ISO 8601 timestamp
</ParamField>

## Response

Returns a paginated list of Pix payment objects.

<ResponseField name="object" type="string">
  Always `list`
</ResponseField>

<ResponseField name="data" type="array">
  Array of Pix payment objects
</ResponseField>

<ResponseField name="has_more" type="boolean">
  Whether more results exist
</ResponseField>

<ResponseField name="total_count" type="integer">
  Total number of payments matching the filters
</ResponseField>

```bash cURL
# List all payments
curl "https://api.bipa.app/v1/pix/payments?limit=20" \

```

```python Python
import requests

# List all completed inbound payments for a customer
response = requests.get(
    "https://api.bipa.app/v1/pix/payments",
    headers={"Authorization": "Bearer sk_live_..."},
    params={
        "customer_id": "cus_a1b2c3d4e5f6",
        "direction": "inbound",
        "status": "completed",
        "limit": 50
    }
)

payments = response.json()
total_received = sum(p["amount_cents"] for p in payments["data"])
print(f"Total received: R$ {total_received / 100:.2f}")
```

## Pagination example

Iterate through all payments for a customer:

```python
def get_all_customer_payments(customer_id):
    payments = []
    starting_after = None

    while True:
        params = {
            "customer_id": customer_id,
            "limit": 100
        }
        if starting_after:
            params["starting_after"] = starting_after

        response = requests.get(
            "https://api.bipa.app/v1/pix/payments",
            headers={"Authorization": "Bearer sk_live_..."},
            params=params
        ).json()

        payments.extend(response["data"])

        if not response["has_more"]:
            break

        starting_after = response["data"][-1]["id"]

    return payments
```

## Aggregating payments

Calculate daily totals:

```python
from collections import defaultdict
from datetime import datetime

payments = get_all_customer_payments("cus_abc123")

daily_totals = defaultdict(lambda: {"inbound": 0, "outbound": 0})

for payment in payments:
    if payment["status"] == "completed":
        date = datetime.fromisoformat(payment["created_at"].replace("Z", "")).date()
        daily_totals[date][payment["direction"]] += payment["amount_cents"]

for date, totals in sorted(daily_totals.items()):
    print(f"{date}: In: R$ {totals['inbound']/100:.2f}, Out: R$ {totals['outbound']/100:.2f}")
```
