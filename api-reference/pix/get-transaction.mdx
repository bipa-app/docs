---
title: "Get Pix Transaction"
description: "Retrieve details of a Pix transaction."
api: "GET https://api.bipa.tech/v1/pix/{id}"
---

## Request

Retrieve details for a specific Pix transaction (inbound or outbound).

### Path parameters

<ParamField path="id" type="string" required>
  The unique transaction identifier (e.g., `pix_abc123`)
</ParamField>

## Response

Returns the full Pix transaction object.

<ResponseField name="id" type="string">
  Unique transaction identifier
</ResponseField>

<ResponseField name="customer_id" type="string">
  Customer associated with this transaction
</ResponseField>

<ResponseField name="direction" type="string">
  `inbound` (received) or `outbound` (sent)
</ResponseField>

<ResponseField name="amount" type="string">
  Transaction amount in centavos
</ResponseField>

<ResponseField name="fee" type="string">
  Fee charged in centavos
</ResponseField>

<ResponseField name="decimals" type="integer">
  Decimal places (always 2 for BRL)
</ResponseField>

<ResponseField name="status" type="string">
  `pending`, `processing`, `completed`, `failed`, `refunded`
</ResponseField>

<ResponseField name="counterparty" type="object" required>
  The other party in the transaction (recipient for outbound, sender for inbound)
  <Expandable title="counterparty">
    <ResponseField name="name" type="string">
      The counterparty's name
    </ResponseField>
    <ResponseField name="document" type="string">
      The counterparty's document (masked)
    </ResponseField>
    <ResponseField name="pix_key" type="string">
      The counterparty's Pix key (if type is `pix_key`)
    </ResponseField>
    <ResponseField name="bank_account" type="object">
      The counterparty's bank account details.
      <Expandable title="bank_account">
        <ResponseField name="ispb_code" type="string">
          The institution ISPB Pix code
        </ResponseField>
        <ResponseField name="branch_number" type="string">
          The branch number
        </ResponseField>
        <ResponseField name="account_number" type="string">
          The account number
        </ResponseField>
        <ResponseField name="account_type" type="string">
          The account type
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="description" type="string">
  Transaction description
</ResponseField>

<ResponseField name="end_to_end_id" type="string">
  Pix network transaction ID
</ResponseField>

<ResponseField name="qr_code_id" type="string">
  Associated QR code (for inbound transactions)
</ResponseField>

<ResponseField name="idempotency_key" type="string">
  Idempotency key (for outbound transactions)
</ResponseField>

<ResponseField name="failure_reason" type="string">
  Error details if status is `failed`
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp
</ResponseField>

<ResponseField name="completed_at" type="string">
  ISO 8601 completion timestamp
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.bipa.tech/v1/pix/pix_abc123 \
  -H "Authorization: Bearer bipa_live_sk_..."
```

```python Python
import requests

response = requests.get(
    "https://api.bipa.tech/v1/pix/pix_abc123",
    headers={"Authorization": "Bearer bipa_live_sk_..."}
)

transaction = response.json()
amount_brl = int(transaction['amount']) / 10 ** transaction['decimals']
print(f"Status: {transaction['status']}, Amount: R$ {amount_brl:.2f}")
```

```javascript Node.js
const response = await fetch(
  "https://api.bipa.tech/v1/pix/pix_abc123",
  {
    headers: { "Authorization": "Bearer bipa_live_sk_..." }
  }
);

const transaction = await response.json();
const amountBrl = Number(transaction.amount) / 10 ** transaction.decimals;
console.log(`Status: ${transaction.status}, Amount: R$ ${amountBrl.toFixed(2)}`);
```
</RequestExample>
