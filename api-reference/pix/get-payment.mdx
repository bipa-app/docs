---
title: "Get Pix Payment"
description: "Retrieve details of a Pix payment."
api: "GET https://api.bipa.app/v1/pix/payments/{payment_id}"
---

## Request

Retrieve details for a specific Pix payment.

### Path parameters

<ParamField path="payment_id" type="string" required>
  The unique payment identifier (e.g., `pix_pay_abc123`)
</ParamField>

## Response

Returns the full Pix payment object.

<ResponseField name="id" type="string">
  Unique payment identifier
</ResponseField>

<ResponseField name="customer_id" type="string">
  Customer associated with this payment
</ResponseField>

<ResponseField name="direction" type="string">
  `inbound` (received) or `outbound` (sent)
</ResponseField>

<ResponseField name="amount_cents" type="integer">
  Payment amount in cents
</ResponseField>

<ResponseField name="fee_cents" type="integer">
  Fee charged
</ResponseField>

<ResponseField name="status" type="string">
  `pending`, `processing`, `completed`, `failed`, `refunded`
</ResponseField>

<ResponseField name="destination" type="object">
  For outbound: recipient details
</ResponseField>

<ResponseField name="payer" type="object">
  For inbound: payer details
</ResponseField>

<ResponseField name="description" type="string">
  Payment description
</ResponseField>

<ResponseField name="end_to_end_id" type="string">
  Pix network transaction ID
</ResponseField>

<ResponseField name="qr_code_id" type="string">
  Associated QR code (for inbound payments)
</ResponseField>

<ResponseField name="idempotency_key" type="string">
  Idempotency key (for outbound payments)
</ResponseField>

<ResponseField name="failure_reason" type="string">
  Error details if status is `failed`
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp
</ResponseField>

<ResponseField name="completed_at" type="string">
  ISO 8601 completion timestamp
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.bipa.app/v1/pix/payments/pix_pay_abc123 \
  -H "Authorization: Bearer sk_live_..."
```

```python Python
import requests

response = requests.get(
    "https://api.bipa.app/v1/pix/payments/pix_pay_abc123",
    headers={"Authorization": "Bearer sk_live_..."}
)

payment = response.json()
print(f"Status: {payment['status']}, Amount: R$ {payment['amount_cents'] / 100:.2f}")
```

```javascript Node.js
const response = await fetch(
  "https://api.bipa.app/v1/pix/payments/pix_pay_abc123",
  {
    headers: { "Authorization": "Bearer sk_live_..." }
  }
);

const payment = await response.json();
console.log(`Status: ${payment.status}, Amount: R$ ${(payment.amount_cents / 100).toFixed(2)}`);
```
</RequestExample>
