## Overview

The Partner API enables B2B integrations with Bipa's infrastructure. Use it to manage Pix keys, track incoming payments, onboard business customers, and process USDT withdrawals.

## Base URL

All Partner API requests are made to:

```
https://api.bipa.tech/partner/v1
```

## Available endpoints

| Resource | Description |
|----------|-------------|
| [Pix Keys](/api-reference/partner/pix-keys/create-pix-key) | Create, retrieve, and delete Pix keys for users |
| [Pix Inflows](/api-reference/partner/pix-in/list-inflows) | Track incoming Pix payments |
| [Events](/api-reference/partner/events/list-events) | Monitor infrastructure events |
| [Customers](/api-reference/partner/customers/create-business) | Onboard business customers |
| [Withdrawals](/api-reference/partner/withdrawals/create-withdrawal) | Process USDT withdrawals |

## Health check

```bash
curl https://api.bipa.tech/partner/v1/health
```
