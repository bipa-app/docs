## Request

Create a new business customer (empresa/PJ) with full KYC information including company details and ownership structure.

### Body parameters

<ParamField body="cnpj" type="string" required>
  Company CNPJ (14 digits). Can be formatted or unformatted.
</ParamField>

<ParamField body="legal_name" type="string" required>
  Registered company name (razão social)
</ParamField>

<ParamField body="trade_name" type="string">
  Doing business as / brand name (nome fantasia)
</ParamField>

<ParamField body="email" type="string" required>
  Company email address
</ParamField>

<ParamField body="phone" type="string" required>
  Phone number in E.164 format (e.g., `+5511999999999`)
</ParamField>

<ParamField body="address" type="object" required>
  Company registered address
  <Expandable title="address">
    <ParamField body="street" type="string" required>
      Street name
    </ParamField>
    <ParamField body="number" type="string" required>
      Street number
    </ParamField>
    <ParamField body="complement" type="string">
      Suite, floor, unit, etc.
    </ParamField>
    <ParamField body="neighborhood" type="string" required>
      Neighborhood (bairro)
    </ParamField>
    <ParamField body="city" type="string" required>
      City name
    </ParamField>
    <ParamField body="state" type="string" required>
      Two-letter state code (e.g., `SP`, `RJ`)
    </ParamField>
    <ParamField body="postal_code" type="string" required>
      CEP (8 digits, numbers only)
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="owners" type="array" required>
  Company owners and representatives. Include all partners with 25% or more ownership.
  <Expandable title="owner object">
    <ParamField body="owner_type" type="string" required>
      Owner type: `partner`, `representative`, `other_partner`
    </ParamField>
    <ParamField body="name" type="string" required>
      Full legal name
    </ParamField>
    <ParamField body="social_name" type="string">
      Social/preferred name
    </ParamField>
    <ParamField body="document" type="string" required>
      CPF number (11 digits)
    </ParamField>
    <ParamField body="email" type="string" required>
      Owner email address
    </ParamField>
    <ParamField body="phone" type="string" required>
      Phone in E.164 format
    </ParamField>
    <ParamField body="birth_date" type="string" required>
      Date of birth in `YYYY-MM-DD` format
    </ParamField>
    <ParamField body="nationality_country_code" type="string" required>
      ISO 3166-1 alpha-2 country code (e.g., `BR`)
    </ParamField>
    <ParamField body="tax_residence_country_code" type="string" required>
      ISO 3166-1 alpha-2 country code for tax residence
    </ParamField>
    <ParamField body="address" type="object" required>
      Owner's residential address
      <Expandable title="address">
        <ParamField body="street" type="string" required>
          Street name
        </ParamField>
        <ParamField body="number" type="string" required>
          Street number
        </ParamField>
        <ParamField body="complement" type="string">
          Apartment, suite, etc.
        </ParamField>
        <ParamField body="neighborhood" type="string" required>
          Neighborhood (bairro)
        </ParamField>
        <ParamField body="city" type="string" required>
          City name
        </ParamField>
        <ParamField body="state" type="string" required>
          Two-letter state code
        </ParamField>
        <ParamField body="postal_code" type="string" required>
          CEP (8 digits)
        </ParamField>
      </Expandable>
    </ParamField>
    <ParamField body="ownership" type="object" required>
      Ownership details
      <Expandable title="ownership">
        <ParamField body="ownership_type" type="string" required>
          Ownership type: `direct` or `indirect`
        </ParamField>
        <ParamField body="ownership_bps" type="integer" required>
          Ownership percentage in basis points (0-10000). Example: 2500 = 25%
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="metadata" type="object">
  Custom key-value pairs for your tracking purposes
</ParamField>

<ParamField body="config" type="object">
  Configuration options
  <Expandable title="config">
    <ParamField body="auto_send_owner_sms" type="boolean" default="false">
      Automatically send SMS verification to owners
    </ParamField>
    <ParamField body="auto_send_owner_email" type="boolean" default="false">
      Automatically send email verification to owners
    </ParamField>
  </Expandable>
</ParamField>

## Ownership calculation

Ownership is specified in **basis points** (bps), where 10000 bps = 100%:

| Percentage | Basis Points |
|------------|--------------|
| 10% | 1000 |
| 25% | 2500 |
| 50% | 5000 |
| 100% | 10000 |

<Warning>
  Include all partners with 25% or more ownership. Failure to disclose significant ownership may result in compliance issues.
</Warning>

## Phone format

Phone numbers must be in E.164 format:

```
+[country code][area code][number]
```

Examples:
- `+5511999999999` (São Paulo mobile)
- `+5521988888888` (Rio de Janeiro mobile)
- `+551133334444` (São Paulo landline)

<Note>
  Business customers undergo compliance verification after creation. Monitor the Events API for `user` events to track when the account becomes active.
</Note>
