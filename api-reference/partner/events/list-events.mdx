---
title: "List Events"
description: "List infrastructure events for your partner account."
---

## Endpoint

```
GET /partner/v1/events
```

## Request

### Headers

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `X-API-Key` | string | Yes | Your partner API key |

### Query parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `delivered` | string | No | Filter by delivery status: `true` or `false` |

## Response

### Success (200 OK)

```json
{
  "events": [
    {
      "id": "evt_abc123",
      "kind": "pix_key",
      "created_at": 1704067200,
      "is_delivered": false,
      "log": {
        "id": "log_xyz789",
        "kind": "registered",
        "created_at": 1704067200,
        "key": "abc123-def456-ghi789",
        "tax_id": "12345678900",
        "timestamp": 1704067200
      }
    }
  ]
}
```

### Event kinds

| Kind | Description |
|------|-------------|
| `pix_key` | Pix key lifecycle events |
| `pix_inflow` | Incoming Pix payment events |
| `user` | User account events |
| `usdt_withdrawal` | USDT withdrawal events |

### Pix Key event log

```json
{
  "id": "log_xyz789",
  "kind": "registered",
  "created_at": 1704067200,
  "key": "abc123-def456-ghi789",
  "tax_id": "12345678900",
  "timestamp": 1704067200
}
```

| Log kind | Description |
|----------|-------------|
| `created` | Pix key creation requested |
| `registered` | Pix key successfully registered |
| `failed` | Pix key registration failed |
| `canceled` | Pix key was deleted |

### USDT Withdrawal event log

```json
{
  "id": "log_xyz789",
  "kind": "settled",
  "created_at": 1704067200,
  "tax_id": "12345678900",
  "amount_usdtmicros": 100000000,
  "blockchain_address": "0x1234...",
  "network": "polygon",
  "tx_hash": "0xabcd...",
  "timestamp": 1704067200

```

| Log kind | Description |
|----------|-------------|
| `created` | Withdrawal request created |
| `sent` | Transaction sent to blockchain |
| `settled` | Transaction confirmed on-chain |
| `failed` | Withdrawal failed |

## Example

```bash
# List all events
curl https://api.bipa.tech/partner/v1/events \
  -H "X-API-Key: your_partner_api_key"

# List only pending events
curl "https://api.bipa.tech/partner/v1/events?delivered=false" \
  -H "X-API-Key: your_partner_api_key"
```
