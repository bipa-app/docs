---
title: "List Events"
description: "Retrieve infrastructure events for your partner integration."
api: "GET https://api.bipa.tech/partner/v1/events"
---

## Request

Retrieve all infrastructure events for your partner. Use the `delivered` query parameter to filter by delivery status.

### Query parameters

<ParamField query="delivered" type="boolean">
  Filter by delivery status:
  - `true` - Only delivered events
  - `false` - Only undelivered events
  - Omit - All events
</ParamField>

## Response

<ResponseField name="events" type="array">
  Array of event objects
  <Expandable title="event object">
    <ResponseField name="id" type="string">
      Unique event identifier (UUID)
    </ResponseField>
    <ResponseField name="kind" type="string">
      Event category: `pix_key`, `pix_inflow`, `user`, `usdt_withdrawal`
    </ResponseField>
    <ResponseField name="created_at" type="integer">
      Unix timestamp when event was created
    </ResponseField>
    <ResponseField name="is_delivered" type="boolean">
      Whether event has been marked as delivered
    </ResponseField>
    <ResponseField name="log" type="object">
      Event-specific payload (structure varies by kind)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.bipa.tech/partner/v1/events?delivered=false \
  -H "X-API-Key: your_partner_api_key"
```

```python Python
import requests

response = requests.get(
    "https://api.bipa.tech/partner/v1/events",
    headers={"X-API-Key": "your_partner_api_key"},
    params={"delivered": "false"}
)

events = response.json()
for event in events["events"]:
    print(f"Event {event['id']}: {event['kind']}")
```
