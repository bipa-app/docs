---
title: "Get Trade"
description: "Retrieve details of a trade."
api: "GET https://api.bipa.app/v1/trades/{trade_id}"
---

## Request

Retrieve details for a specific trade.

### Path parameters

<ParamField path="trade_id" type="string" required>
  The unique trade identifier (e.g., `trd_abc123xyz`)
</ParamField>

## Response

Returns the full trade object.

<ResponseField name="id" type="string">
  Unique trade identifier
</ResponseField>

<ResponseField name="customer_id" type="string">
  Customer who executed the trade
</ResponseField>

<ResponseField name="quote_id" type="string">
  Associated quote ID (if executed via quote)
</ResponseField>

<ResponseField name="pair" type="string">
  Trading pair
</ResponseField>

<ResponseField name="side" type="string">
  `buy` or `sell`
</ResponseField>

<ResponseField name="status" type="string">
  `pending`, `completed`, `failed`, `cancelled`
</ResponseField>

<ResponseField name="from_currency" type="string">
  Currency debited
</ResponseField>

<ResponseField name="to_currency" type="string">
  Currency credited
</ResponseField>

<ResponseField name="from_amount_cents" type="integer">
  Amount debited in cents (for BRL)
</ResponseField>

<ResponseField name="from_amount" type="string">
  Amount debited (for crypto)
</ResponseField>

<ResponseField name="to_amount_cents" type="integer">
  Amount credited in cents (for BRL)
</ResponseField>

<ResponseField name="to_amount" type="string">
  Amount credited (for crypto)
</ResponseField>

<ResponseField name="executed_rate" type="string">
  Final execution rate
</ResponseField>

<ResponseField name="fee_cents" type="integer">
  Fee charged in BRL cents
</ResponseField>

<ResponseField name="fee_currency" type="string">
  Fee currency
</ResponseField>

<ResponseField name="failure_reason" type="string">
  Error details if status is `failed`
</ResponseField>

<ResponseField name="idempotency_key" type="string">
  Idempotency key used
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp
</ResponseField>

<ResponseField name="completed_at" type="string">
  ISO 8601 timestamp
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.bipa.app/v1/trades/trd_abc123xyz \
  -H "Authorization: Bearer sk_live_..."
```

```python Python
import requests

response = requests.get(
    "https://api.bipa.app/v1/trades/trd_abc123xyz",
    headers={"Authorization": "Bearer sk_live_..."}
)

trade = response.json()
print(f"Trade {trade['id']}: {trade['status']}")
print(f"Converted {trade['from_amount_cents']/100} BRL to {trade['to_amount']} USDT")
```

```javascript Node.js
const response = await fetch(
  "https://api.bipa.app/v1/trades/trd_abc123xyz",
  {
    headers: { "Authorization": "Bearer sk_live_..." }
  }
);

const trade = await response.json();
console.log(`Trade ${trade.id}: ${trade.status}`);
```
</RequestExample>
