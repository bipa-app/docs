---
title: "List Customers"
description: "List all customers with pagination and filtering."
api: "GET https://api.bipa.app/v1/customers"
---

## Request

Retrieve a paginated list of customers.

### Query parameters

<ParamField query="limit" type="integer" default="20">
  Number of customers to return (1-100)
</ParamField>

<ParamField query="starting_after" type="string">
  Cursor for pagination. Use the `id` of the last customer from the previous page.
</ParamField>

<ParamField query="ending_before" type="string">
  Cursor for reverse pagination.
</ParamField>

<ParamField query="type" type="string">
  Filter by customer type: `individual` or `business`
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `pending`, `active`, `suspended`, `rejected`, `inactive`
</ParamField>

<ParamField query="email" type="string">
  Filter by exact email match
</ParamField>

<ParamField query="created_after" type="string">
  Filter customers created after this ISO 8601 timestamp
</ParamField>

<ParamField query="created_before" type="string">
  Filter customers created before this ISO 8601 timestamp
</ParamField>

## Response

Returns a paginated list of customers.

<ResponseField name="object" type="string">
  Always `list`
</ResponseField>

<ResponseField name="data" type="array">
  Array of customer objects
</ResponseField>

<ResponseField name="has_more" type="boolean">
  Whether more results exist
</ResponseField>

<ResponseField name="total_count" type="integer">
  Total number of customers matching the filters
</ResponseField>

<RequestExample>
```bash cURL
# List all customers
curl "https://api.bipa.app/v1/customers?limit=20" \
  -H "Authorization: Bearer sk_live_..."

# Filter by type and status
curl "https://api.bipa.app/v1/customers?type=individual&status=active" \
  -H "Authorization: Bearer sk_live_..."

# Paginate with cursor
curl "https://api.bipa.app/v1/customers?limit=20&starting_after=cus_abc123" \
  -H "Authorization: Bearer sk_live_..."
```

```python Python
import requests

# List all customers
response = requests.get(
    "https://api.bipa.app/v1/customers",
    headers={"Authorization": "Bearer sk_live_..."},
    params={"limit": 20}
)

# Filter by type
response = requests.get(
    "https://api.bipa.app/v1/customers",
    headers={"Authorization": "Bearer sk_live_..."},
    params={
        "type": "individual",
        "status": "active"
    }
)

customers = response.json()
```

## Pagination example

Iterate through all customers:

```python
def get_all_customers():
    customers = []
    starting_after = None

    while True:
        params = {"limit": 100}
        if starting_after:
            params["starting_after"] = starting_after

        response = requests.get(
            "https://api.bipa.app/v1/customers",
            headers={"Authorization": "Bearer sk_live_..."},
            params=params
        ).json()

        customers.extend(response["data"])

        if not response["has_more"]:
            break

        starting_after = response["data"][-1]["id"]

    return customers
```
