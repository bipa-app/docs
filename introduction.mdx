## Quick example

Create a customer and execute a BRL to USDT trade:

<CodeGroup>

```bash cURL
# Create a customer
curl -X POST https://api.bipa.app/v1/customers \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{
    "type": "individual",
    "document": {"type": "cpf", "number": "12345678900"},
    "name": "João Silva",
    "email": "joao@example.com"
  }'

# Get a quote
curl "https://api.bipa.app/v1/quotes?from=BRL&to=USDT&amount_cents=100000" \
  -H "Authorization: Bearer sk_live_..."

# Execute the trade
curl -X POST https://api.bipa.app/v1/quotes/quote_abc123/execute \
  -H "Authorization: Bearer sk_live_..." \
  -H "Content-Type: application/json" \
  -d '{"customer_id": "cus_a1b2c3d4e5f6"}'
```

```python Python
import requests

headers = {
    "Authorization": "Bearer sk_live_...",
    "Content-Type": "application/json"
}

# Create a customer
customer = requests.post(
    "https://api.bipa.app/v1/customers",
    headers=headers,
    json={
        "type": "individual",
        "document": {"type": "cpf", "number": "12345678900"},
        "name": "João Silva",
        "email": "joao@example.com"
    }
).json()

# Get a quote
quote = requests.get(
    "https://api.bipa.app/v1/quotes",
    headers=headers,
    params={"from": "BRL", "to": "USDT", "amount_cents": 100000}
).json()

# Execute the trade
trade = requests.post(
    f"https://api.bipa.app/v1/quotes/{quote['id']}/execute",
    headers=headers,
    json={"customer_id": customer["id"]}
).json()
```

```javascript Node.js
const headers = {
  "Authorization": "Bearer sk_live_...",
  "Content-Type": "application/json"
};

// Create a customer
const customer = await fetch("https://api.bipa.app/v1/customers", {
  method: "POST",
  headers,
  body: JSON.stringify({
    type: "individual",
    document: { type: "cpf", number: "12345678900" },
    name: "João Silva",
    email: "joao@example.com"
  })
}).then(r => r.json());

// Get a quote
const quote = await fetch(
  "https://api.bipa.app/v1/quotes?from=BRL&to=USDT&amount_cents=100000",
  { headers }
).then(r => r.json());

// Execute the trade
const trade = await fetch(
  `https://api.bipa.app/v1/quotes/${quote.id}/execute`,
  {
    method: "POST",
    headers,
    body: JSON.stringify({ customer_id: customer.id })
  }
).then(r => r.json());
```
